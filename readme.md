# Yu-Gi-Oh | Jo-ken-po Edition

## Sobre o Projeto

Jogo interativo de Pedra, Papel e Tesoura com temática de Yu-Gi-Oh, desenvolvido para demonstrar conceitos fundamentais de lógica de programação aplicados ao desenvolvimento de jogos.

## Conceitos Abordados

- **Armazenamento e Gerenciamento de Estado Manual**: Controle e manipulação do estado do jogo sem uso de frameworks
- **Funções Limpas**: Implementação de código modular e reutilizável seguindo boas práticas
- **Organização de Código**: Estruturação lógica e manutenível do projeto
